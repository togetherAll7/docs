# ToggleButton

You can turn it into a button with aria-pressed by using the role togglebutton, you'll then be able to use the following properties deactivationMsg in addition to the usual description and activationMsg properties.

```jsx
<A11y
  role="togglebutton"
  description="Dark theme "
  activationMsg="Switched to dark theme"
  deactivationMsg="Switched to light theme" ... />
```

Special attributes: `deactivationMsg`
